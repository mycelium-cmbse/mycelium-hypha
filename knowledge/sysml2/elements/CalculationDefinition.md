# CalculationDefinition

`Calculations` package · concrete metaclass

A CalculationDefinition is an <coed>ActionDefinition that also defines a Function producing a result.</coed>

## Generalizations

- [ActionDefinition](ActionDefinition.md)

## Owned features

### calculation : CalculationUsage [0..*]

The actions of this CalculationDefinition that are CalculationUsages.
